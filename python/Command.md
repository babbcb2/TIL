### sequence 

* **`list`**

  * 임의의 값을 순서대로 저장하는 집합 자료형 (index 부여 및 값 변경가능한 형식)

  * 함수를 이용하는 방법과 [ ] 를 이용하는 방법이 있다.

  * range () 이용하여 리스트를 생성하는 방법이 있다.

    ```
    a = []
    print (type(a))
    ==> <class 'list'>
    
    a = list()
    print (type(a))
    ==> <class 'list'>
    
    a = [1,2,3]
    print (a)
    ==> [1,2,3]
    ```

* **`tuple`**

  ```
  a = ()
  print(type(a))
  ==> <class 'tuple'>
  ```

* indexing

  ```
  print (a[0])
  print (a[1])
  print (a[2])
  print (a[-1])
  ```

* slicing

  ```
  print(a[0:2])
  ==> [1, 2]
  
  a = [1, 2, 'hello', 3.14]
  a = [1, 2, ['show','me','the','money'], 3.14]
  print(type(a[2]))
  ==> <class 'list'>
  
  print(a[2][2:])
  ==> ['the', 'money']
  
  a = [1,2,3]
  b = [4,5,6]
  print(a+b)
  ==> [1, 2, 3, 4, 5, 6]
  print(a*3)
  ==> [1, 2, 3, 1, 2, 3, 1, 2, 3]
  
  print('p','y','t','h','o','n')
  ==> p y t h o n
  print('p','y','t','h','o','n', sep="")
  ==> python
  print('p','y','t','h','o','n', sep="-")
  ==> p-y-t-h-o-n
  ```
